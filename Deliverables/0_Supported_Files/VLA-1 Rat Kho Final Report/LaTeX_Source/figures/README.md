# Figure Support Files

This directory contains the figure assets and figure source files used by `../paper.tex`. It is kept under `Deliverables/0_Supported_Files/` so the numbered deliverable folders only hold final exported outputs.

## Current files

- `system_architecture.png`
- `dashboard_screenshot.png`
- `prompt_pipeline.png`
- `system_architecture.tex`
- `prompt_pipeline.tex`

## Usage

- Keep the filenames stable so `paper.tex` continues to compile without path changes.
- Replace the PNGs with higher-quality exports if you refresh the figures for the report.

## Recommended replacement workflow

1. Export each final figure in the filename already referenced by `paper.tex`.
2. Save it into this directory using the expected filename.
3. Recompile `paper.tex`.

No changes to the LaTeX source are required if the filenames stay the same.
